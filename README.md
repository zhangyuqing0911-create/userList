今回開発した画面に実装した機能  
ユーザリスト  
・左側のリスト画面に二つの形態を実装。→アイコンリスト形態/ユーザリスト形態。ヘッダアイコン押下で切り替え可能。  
・アイコン形態でタップするとユーザリスト形態になり、ユーザリスト形態でクリックしたユーザの詳細ページが右側に表示されます。  

詳細ページ  
・詳細ページではフォロー/フォロワーリスト、自分発信イベント/受信イベントリスト、作品リストの５つのリストが展開されます。  

フォロー/フォロワーリスト  
・フォロー中/フォロワーの文字列をクリックするとフォロー/フォロワーリストが開かれます。  
・ヘッダ部のボタンをクリックすることでリストの切り替え可能。  
・左上の←をクリックして（元のユーザの）詳細画面に戻ります。  
・アイコンをクリックして（クリックしたユーザの）詳細画面を開きます。  

自分/受信/作品タブ  
・自分/受信/作品タブの文字列をクリックすると表示リストが切り替わります。  
・自分/受信タブにてイベントプルダウンをクリックすると、取得したリストのイベント種類が表示され、選択後フィルターします。  
・自分/受信タブにて昇順/降順をクリックし日付で並べ替えます。  
・自分/受信/作品タブの文字列をクリックすると検索条件がリセットされます。  
・イベント内容のリポジトリURLはリンクです。  
・アイコンをクリックして（クリックしたユーザの）詳細画面を開きます。  

こだわりポイント  
・タブやボタンを押したとき、下線部をけして下の画面とつながったようにみせて、視覚的にわかりやすくしました。  
・フォロー中をクリックしたときはフォローリストを選択した状態で、フォロワーをクリックしたときはフォロワーリストを選択した状態で開き、ストレスを少なくしました。  
・自分/受信/作品リストは詳細画面ロード時に取得、フォロー/フォロワーリストはフォロー中/フォロワークリック時にすることで、リアルタイムな増減を心掛けました。  

改善点  
・イベントの種類が多かったため、個別な対応ができなかったため、種類ごとに詳細を表示できるようにしたい。  
・クリックやその他の動きに滑らかさが足りない。スライドなどのアニメーションがあったらよかった。  
・アイコン状態でスクロールし、ユーザリスト状態に切り替えると、一番上まで戻ってしまう。そのまま表示出来たらよい。  
・デザインが活動内容を見るのに向いていない  

新規で足したい機能  
・直近３０日に更新があった作品など、活動内容を分かりやすくする  
・ユーザを検索できるようにする。
